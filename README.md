## Prerequisites

In order to compile and run the **email2sms** server you need to have [Erlang](http://www.erlang.org/) and GNU make installed.

## Compilation

<pre>
$ git clone https://github.com/PowerMeMobile/emal2sms.git
$ cd email2sms
$ make
</pre>

## Usage

<pre>

$ make console

</pre>
